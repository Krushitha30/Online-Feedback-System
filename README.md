# Online Feedback System

## Project Description
This is an Online Feedback System developed using Flask and SQLite.
It allows students to submit feedback for subject teachers online instead of manual forms.

## Technologies Used
- Python (Flask)
- SQLite
- HTML
- CSS
- JavaScript

## How to Run the Project
1. Clone the repository
2. Run `python init_db.py`
3. Run `python app.py`
4. Open browser and go to `http://127.0.0.1:5000/`

## Features
- Student Login
- Admin Login
- Feedback Submission
- Duplicate Feedback Prevention
- Admin Feedback View
